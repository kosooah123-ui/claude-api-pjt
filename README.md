# 냉장고 레시피 추천 웹앱

냉장고 사진을 업로드하면 AI가 식재료를 인식하고, 인식된 재료로 만들 수 있는 레시피를 추천해주는 웹 애플리케이션입니다.

## 진행 단계

전체 기능은 3단계로 나누어 구현되어 있으며, 각 단계의 상세 요구사항은 `docs/` 폴더의 PRD 문서를 참고하세요.

1. **[PRD_01](docs/PRD_01.md) — 냉장고 사진 인식**: 이미지 업로드 → Vision 모델로 식재료 인식 → 목록 반환
2. **[PRD_02](docs/PRD_02.md) — 레시피 생성**: 인식된 식재료 + 사용자 선호도 → 텍스트 모델로 레시피 추천
3. **[PRD_03](docs/PRD_03.md) — 사용자 프로필 및 저장**: 회원가입/로그인, 레시피 저장/조회/삭제

## 기술 스택

- Node.js / Express
- OpenRouter API (`nvidia/nemotron-nano-12b-v2-vl:free` — 이미지 인식 및 레시피 생성 모두 사용)
- JSON 파일 기반 저장소 (`data/db.json`, 프로토타입 단계)
- 순수 HTML/CSS/JS 프론트엔드 (`public/`)

## 실행 방법

```bash
npm install
```

`.env` 파일에 OpenRouter API 키를 설정합니다 (`.env.example` 참고):

```
OPENROUTER_API_KEY=your_key_here
```

서버 실행:

```bash
npm start
```

브라우저에서 `http://localhost:3000` 접속.

## 알려진 제한사항

- 사용 중인 모델(`nemotron-nano-12b-v2-vl:free`)은 무료 소형 모델이라 이미지 인식/레시피 생성 결과에 환각(존재하지 않는 재료, 부자연스러운 문장)이 섞일 수 있습니다. 사용자가 인식 결과를 직접 수정할 수 있는 UI를 제공합니다.
- 세션 토큰은 서버 메모리에만 저장되어 서버 재시작 시 모든 로그인이 초기화됩니다.
- 데이터 저장소가 JSON 파일 기반이라 다수 사용자 동시 접속에는 적합하지 않습니다.
